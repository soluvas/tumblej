Thank you for downloading TumbleJ. TumbleJ is a Java API for Tumblr for posting and retrieving posts. File uploads are not supported yet. Audio and video posts with external links can be uploaded. 

Changes between 1.0 and 1.1:

- Total rewrite using packages 
- Most of the methods not backward compatible
- See Javadocs for more details
